Lidar

Par Nico

Lidar est un module de gestion du Lidar 2D base sur un modele SICK TIM561

Le module gere plusieurs fonctionnalites :

  • Communication avec le Lidar reel
  • Enregistrement des donnees brutes dans un fichier texte au format .csv
  • Rejeu a posteriori d'un fichier de donnees
  • Affichage des donnes sur graphe polaire ou lineaire (donnees du capteur reel ou du rejeu)
  • Filtrage des donnees brutes avec reglages, sauvegarde et restitution des parametres du filtre (plusieurs filtres possibles)


Paramètres de configuration pour la communication avec le capteur SICK TIM561

  • sick_port : numéro du port de communication
  • sick_ip : adresse IP du SICK TIM561
  • lidar_sample_period : periode de lecture des infos du capteur [msec]
  • sick_autoreconnect: reconnexion automatique au capteur SICK en cas de perte de communication


Paramètres de configuration pour l'affichage

  • data_displayed : choix des donnees affichees sur le graphe (brutes ou filtrees)
  • graph_type : type de graphique affiche (polaire ou lineaire)
  • active_graph : active ou inhibe l'affichage graphique


Paramètres de configuration pour le data logger

  • logger_refresh_period : periode de sauvegarde des donnees (!! PAS ENCORE IMPLEMENTE !!)

=> Dans l'etat actuel, la periode de log est la meme que la periode de reception des donnees


Paramètres de configuration pour le filtrage des donnees

  • index_data_filter : numero du dernier filtre utilise
  • lidar_params_eeprom_filename : nom complet (avec chemin) du fichier EEPROM contenant les parametres des differents filtres (par defaut : ./Config/lidar_filter_params.ini)
