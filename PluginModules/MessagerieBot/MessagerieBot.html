MessagerieBot

Par Nico

Le module MessageriBot gère les messages échangés avec le robot.

En réception, il reçoit les octets bruts de la RS232, décode, compabilise les trames.

En émission, il encode les octets de la trame et envoie vers la RS232.


Un diagnostic de communication avec le robot est réalisé.

Le module indique si la connexion avec le robot est établie ou perdue.

La communication est considérée établie (connecté) lorsqu'une trame valide est reçue du robot.

Elle est considérée perdue (déconnecté) lorsqu'aucune trame valide n'est reçue depuis un certain temps.


Une donnée Robot_Connecte de type bool est mise à jour dans le data center pour refléter la connexion.

Il est également possible de s'abonner au signal connected(bool state) émis par le module.
