XBEE

Par Nico

Ce plugin permet de :

  • Configurer un module XBEE avec l'ensemble de ses paramètres réseaux
  • Analyser le traffic vers et en provenance du XBEE
  • Envoyer des données vers le XBEE


Un point d'entrée (slot) pour encapsuler et envoyer des données vers le XBEE : encode(data, destination_address)

Un point de sortie (signal) pour récupérer les données valides reçues du XBEE : readyBytes(data, source_address)

Pour les données sortantes, destination_address représente l'ID du module XBEE destinataire du message (0xFFFF pour un message de broadcast).

Pour les données entrantes, source_address représente l'ID du module XBEE émetteur du message.
