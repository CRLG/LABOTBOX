UserGuides

Par Nico


UserGuides est le module d'affichage de l'aide en ligne sur les modules.

Chaque module peut avoir sa propre documentation au format html.


La documentation est intégrée dans l'exécutable LaBotBox par le mécanisme de ressources.

Lorsque la documentation est modifiée, il est nécessaire de recompiler LaBotBox voir apparaitre les modifications.


Une documentation vierge est automatiquement créée lorsq'un module est créé avec ModuleDesigner.


Pour ajouter manuellement de la documentation à un module :

  • Créer dans le répertoire du module un fichier NomModule.html contenant la documentation du module.
  • S'il n'existe pas déjà, créer dans le répertoire du module un fichier de ressources NomModule.qrc
  • Ajouter dans ce fichier de ressource la ligne :

<qresource prefix="/user_guide">

<file>NomModule.html</file>

</qresource>


LaBotBox intègre également un éditeur html pour créer ou modifier ces documentations.

Cet éditeur est accessible depuis le menu "Edit" -> "Html Text Editor"
