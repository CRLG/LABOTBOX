Joystick

Par Nico

Le module Joystick permet de mettre à disposition les états d'un joystick connecté dans le DataManager.

Les données se nomment :

  • Joystick<x>_Connected pour indiquer si le joystick est connecté ou non (valeur 0="Non connecté" / 1="Connecté")
  • Joystick<x>_Axis<y> pour les axes (valeur entre 0.0 et 100.0)
  • Joystick<x>_Button<z> pour les appuis pushs (valeur 0="relâché" / 1="appuyé")

où :

  • x [0;1] représente le numéro du joystic (le module gère 2 joysticks possibles)
  • y [0;7] représente le numéro de l'axe (8 axes possibles par jostick)
  • z [0;11] représente le numéro du boutton (12 bouttons possibles par joystick)

Les valeurs ci-dessus pour x, y, z peuvent être modifiées et ont pour limite les possibilités de la librairies sfml (la lib peut gérer jusqu'à sf::Joystick::Count joysticks différents)
